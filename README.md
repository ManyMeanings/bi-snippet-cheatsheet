## 使用方法

在 VSCODE 中依次点击设置-用户代码片段-新建全局代码片段文件，将 work.code-snippets 文件内所有内容复制进去。

在页面中键入代码片段对应的 prefix，选择后用 tab 键依次切换占位符。
